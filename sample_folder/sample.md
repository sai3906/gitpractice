Navigate to your repository on GitHub.

Click Add file > Create new file.

In the file name box, type the folder name you want, then type a forward slash (/).

    Example: Type my-new-folder/

    The interface will instantly convert my-new-folder into a directory structure.

Type the name of a dummy file (usually .gitkeep or README.md) inside that folder.

Add some placeholder text in the file content, scroll down, and click Commit changes.
